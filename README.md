# FROJAX FREE CLICK (Final)

Estructura lista para GitHub Pages.

## Pasos
1. Crea repo (p.ej. `frojax-free-click`).
2. Sube todos los archivos de este ZIP (mantén `assets/` junto a `index.html`).
3. Settings → Pages → Deploy from a branch → `main` → `/`.
4. Abre la URL de Pages.

## Notas
- Botón PLAY único (play/pause).
- Pantalla parpadea por pulso (negro).
- BPM en dorado, perillas de slider en dorado.
- Atajos: 75–200 en saltos de 25 (sin 50).
- “Sonido / Notas por pulso” en una sola línea.
- Footer responsivo con logos en extremos y links centrados (INSTA, MENTORÍAS / LECCIONES, FROJAX.COM).
- Modo entrenador muestra panel al activarse, no inicia reproducción por sí mismo.
